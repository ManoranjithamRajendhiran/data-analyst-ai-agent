# 📊 Data Analyst AI Agent

An intelligent **AI-powered Data Analyst Agent** that automates end-to-end data analysis using natural language — from data cleaning to generating actionable insights.

---

## 🎯 Why I Built This

As an AI & Data Science student, I wanted to solve a real problem:

> “Data analysis requires technical skills — what if anyone could analyze data just by asking questions?”

This project bridges that gap by combining:
- AI Agents
- Data Analysis
- Automation

---

## 🚀 Features

- 🧠 Natural Language Query Processing  
- 📂 Dataset Handling (CSV / Excel)  
- 🧹 Automated Data Cleaning  
- 📊 Exploratory Data Analysis (EDA)  
- 📈 Data Visualization  
- 🔍 Insight Generation  
- ⚡ Multi-step reasoning  

---

## ⚙️ Tech Stack

**🧠 AI / Agent Layer**

**📊 Data Processing**
